# Mutual_Fund_Analysis
Overview

This Jupyter Notebook is designed to analyze and visualize mutual fund data. It walks through loading the dataset, cleaning it, performing exploratory data analysis (EDA), and creating interactive visualizations using Plotly Express.

Main Sections

Introduction: A brief overview of the notebook’s goals.

Data Loading: Loads mutual fund data from a CSV file.

Data Cleaning: Preprocesses the data to ensure it’s ready for analysis.

Exploratory Data Analysis (EDA): Uses Plotly Express to create interactive visualizations and explore the data.

Performance Analysis: Compares mutual funds based on performance metrics.

Key Outputs

Summary statistics of mutual fund performance.

Interactive Plotly Express visualizations that highlight trends and comparisons.

Requirements and Setup

Python Libraries: The notebook uses pandas for data manipulation and plotly.express for interactive visualizations.

Installation: Ensure you have these libraries installed via pip: pip install pandas plotly.
